# Portfolio de 15 Landing Pages

Este proyecto es una colección de 15 *landing pages* (páginas de aterrizaje) construidas desde cero utilizando **HTML5** y **CSS3**. El objetivo principal de este proyecto es la práctica y el dominio del diseño web responsivo, la maquetación moderna con Flexbox y CSS Grid, y la atención al detalle visual (UI/UX).

## 🚀 Tecnologías Utilizadas

- **HTML5:** Estructuración semántica del contenido.
- **CSS3:** Estilos avanzados, animaciones, gradientes, y efectos visuales (como *glassmorphism*).
- **Flexbox & CSS Grid:** Para la maquetación de layouts complejos y alineaciones precisas.
- **Diseño Responsivo (Media Queries):** Adaptación perfecta a diferentes tamaños de pantalla (móvil, tablet, escritorio).

## 📂 Estructura del Proyecto

El proyecto está organizado de la siguiente forma:

```text
15-landing pages/
├── landing-pages/
│   ├── style.css           # Estilos principales para el índice (modo oscuro premium)
│   ├── html/               # Contiene los archivos .html individuales de cada landing page
│   ├── css/                # Contiene los archivos .css específicos de cada landing page
│   └── image/              # Activos gráficos e imágenes clasificadas por página
└── index.html              # Página principal con el índice interactivo de todas las landing pages
└── README.md               # Este archivo
```

## 📋 Páginas Incluidas

El índice interactivo (`index.html`) proporciona acceso directo a las siguientes 15 páginas desarrolladas:

10. **Página 33:** Diseño dinámico de presentación de productos con grillas superpuestas.

13. **Página 41:** Diseño avanzado con textos superpuestos en SVG estéticos.
14. **Página 45:** Maquetación conceptual "Grid 2x2" oscura y elegante.
15. **Página 50:** Composición visual en bloques tipo mosaico (masonry/bento grid).

## 💻 Cómo ejecutar el proyecto

No se requiere ningún marco de trabajo (*framework*) o servidor especial para visualizar este proyecto.

1. Clona o descarga el repositorio en tu dispositivo.
2. Abre la carpeta del proyecto.
3. Haz doble clic en el archivo `landing-pages/index.html` para abrirlo en tu navegador web predeterminado.
4. Desde el índice, puedes navegar y probar el diseño responsivo de cada una de las 15 landing pages.
